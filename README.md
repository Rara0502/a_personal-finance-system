# 个人记账软件

这是一个基于Python开发的个人记账软件，提供收支管理、预算设置、统计分析等功能。

## 功能特性

- 用户注册与登录
- 收入支出记录管理
- 分类管理（支持自定义分类）
- 月度预算设置与超支提醒
- 收支统计分析（日、月、年统计）
- 交易记录搜索
- 数据导出功能

## 技术栈

- Python 3.6+
- Tkinter (GUI库)
- SQLite3 (数据库)

## 项目结构

```
code/
├── database.py    # 数据库管理模块

├── statistics.py  # 统计分析模块
```

## 运行方法

1. 确保已安装Python 3.6或更高版本

2. 直接运行主程序：
   ```
   python main.py
   ```
   或者在Windows系统中双击main.py文件

## 使用说明

1. **注册登录**
   - 首次使用需要先注册账号
   - 登录后进入主界面

2. **记账功能**
   - 点击"快速记账"或"添加记账"按钮
   - 选择交易类型（收入/支出）
   - 输入金额、选择分类、填写日期时间和备注
   - 点击保存完成记账

3. **预算设置**
   - 点击"预算管理"按钮
   - 设置默认月度预算和当前月预算
   - 系统会自动计算已花费金额和剩余预算

4. **分类管理**
   - 点击"分类管理"按钮
   - 可以添加或删除自定义分类
   - 预设分类不能删除

5. **统计分析**
   - 通过"统计"菜单选择不同统计类型
   - 查看收支总览和分类统计

6. **数据导出**
   - 通过"文件"菜单或快捷按钮导出数据
   - 数据将以JSON格式保存在程序目录下

## 注意事项

- 数据存储在本地SQLite数据库中
- 密码使用SHA-256进行加密存储
- 为保证数据安全，建议定期导出备份数据
- 首次运行时会自动创建必要的数据表和默认分类
